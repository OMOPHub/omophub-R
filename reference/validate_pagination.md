# Validate Page Parameters

Validate Page Parameters

## Usage

``` r
validate_pagination(
  page,
  page_size,
  max_page_size = 1000,
  call = rlang::caller_env()
)
```

## Arguments

- page:

  Page number.

- page_size:

  Page size.

- max_page_size:

  Maximum allowed page size.

- call:

  The calling environment.
